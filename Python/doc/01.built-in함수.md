# Python Built-in 함수 강의안

## 1. Built-in 함수란?

Python의 Built-in 함수는 별도의 `import` 없이 바로 사용할 수 있는 기본 함수이다.

예를 들어 다음 함수들은 모두 Built-in 함수이다.

```python
print()
input()
int()
str()
len()
type()
range()
sum()
max()
min()
```

---

# 2. 입출력 관련 함수

## 2.1 print()

값을 화면에 출력한다.

```python
print("Hello Python")
print(100)
print("점수:", 95)
```

실행 결과

```text
Hello Python
100
점수: 95
```

---

## 2.2 input()

사용자로부터 문자열을 입력받는다.

```python
name = input("이름 입력: ")
print("안녕하세요", name)
```

주의할 점은 `input()`의 결과는 항상 문자열이다.

```python
age = input("나이 입력: ")
print(type(age))
```

---

# 3. 자료형 변환 함수

## 3.1 int()

문자열이나 실수를 정수로 변환한다.

```python
num = int("100")
print(num + 10)
```

```python
num = int(3.14)
print(num)
```

실행 결과

```text
3
```

---

## 3.2 float()

문자열이나 정수를 실수로 변환한다.

```python
height = float("175.5")
print(height)
```

---

## 3.3 str()

값을 문자열로 변환한다.

```python
age = 20
text = str(age)

print(text)
print(type(text))
```

---

## 3.4 bool()

값을 참 또는 거짓으로 변환한다.

```python
print(bool(1))
print(bool(0))
print(bool(""))
print(bool("Python"))
```

실행 결과

```text
True
False
False
True
```

---

# 4. 자료형 확인 함수

## 4.1 type()

값의 자료형을 확인한다.

```python
print(type(100))
print(type(3.14))
print(type("Python"))
print(type([1, 2, 3]))
```

---

## 4.2 isinstance()

특정 자료형인지 확인한다.

```python
num = 10

print(isinstance(num, int))
print(isinstance(num, str))
```

실행 결과

```text
True
False
```

---

# 5. 길이와 범위 관련 함수

## 5.1 len()

문자열, 리스트, 튜플 등의 길이를 구한다.

```python
text = "Python"
print(len(text))

numbers = [10, 20, 30]
print(len(numbers))
```

---

## 5.2 range()

연속된 숫자 범위를 만든다.

```python
for i in range(5):
    print(i)
```

실행 결과

```text
0
1
2
3
4
```

```python
for i in range(1, 6):
    print(i)
```

```python
for i in range(1, 10, 2):
    print(i)
```

실행 결과

```text
1
3
5
7
9
```

---

# 6. 수학 계산 관련 함수

## 6.1 sum()

전체 합계를 구한다.

```python
scores = [80, 90, 70]
print(sum(scores))
```

---

## 6.2 max()

가장 큰 값을 구한다.

```python
scores = [80, 90, 70]
print(max(scores))
```

---

## 6.3 min()

가장 작은 값을 구한다.

```python
scores = [80, 90, 70]
print(min(scores))
```

---

## 6.4 abs()

절댓값을 구한다.

```python
print(abs(-10))
print(abs(10))
```

---

## 6.5 round()

반올림한다.

```python
print(round(3.14))
print(round(3.56))
print(round(3.14159, 2))
```

실행 결과

```text
3
```

---

## 6.6 pow()

거듭제곱을 계산한다.

```python
print(pow(2, 3))
```

실행 결과

```text
8
```

`**` 연산자로도 가능하다.


---

## 6.7 divmod()

몫과 나머지를 동시에 구한다.

```python
result = divmod(10, 3)
print(result)
```

실행 결과

```text
(3, 1)
```

```python
q, r = divmod(10, 3)

print("몫:", q)
print("나머지:", r)
```

# 7. 정렬 관련 함수

## 7.1 sorted()

정렬된 새로운 리스트를 반환한다.

```python
numbers = [3, 1, 5, 2, 4]

result = sorted(numbers)

print(result)
print(numbers)
```

실행 결과

```text
[1, 2, 3, 4, 5]
[3, 1, 5, 2, 4]
```

내림차순 정렬

```python
numbers = [3, 1, 5, 2, 4]

result = sorted(numbers, reverse=True)

print(result)
```

---

## 7.2 reversed()

순서를 뒤집는다.

```python
numbers = [1, 2, 3, 4, 5]

result = list(reversed(numbers))

print(result)
```

---

# 8. 반복 처리 관련 함수

## 8.1 enumerate()

반복문에서 인덱스와 값을 함께 꺼낸다.

```python
names = ["Kim", "Lee", "Park"]

for index, name in enumerate(names):
    print(index, name)
```

실행 결과

```text
0 Kim
1 Lee
2 Park
```

시작 번호 변경

```python
names = ["Kim", "Lee", "Park"]

for index, name in enumerate(names, start=1):
    print(index, name)
```

---

## 8.2 zip()

여러 리스트를 묶어서 처리한다.

```python
names = ["Kim", "Lee", "Park"]
scores = [90, 80, 70]

for name, score in zip(names, scores):
    print(name, score)
```

실행 결과

```text
Kim 90
Lee 80
Park 70
```

---

## 8.3 map()

각 요소에 함수를 적용한다.

```python
numbers = ["10", "20", "30"]

result = list(map(int, numbers))

print(result)
```

실행 결과

```text
[10, 20, 30]
```

입력 처리에서 자주 사용한다.

```python
a, b = map(int, input().split())

print(a + b)
```

---

## 8.4 filter()

조건에 맞는 값만 걸러낸다.

```python
numbers = [1, 2, 3, 4, 5, 6]

def is_even(n):
    return n % 2 == 0

result = list(filter(is_even, numbers))

print(result)
```

실행 결과

```text
[2, 4, 6]
```

---

# 9. 논리 판단 함수

## 9.1 all()

모든 값이 참이면 `True`를 반환한다.

```python
values = [True, True, True]
print(all(values))
```

```python
scores = [80, 90, 70]

result = all(score >= 60 for score in scores)

print(result)
```

---

## 9.2 any()

하나라도 참이면 `True`를 반환한다.

```python
values = [False, False, True]
print(any(values))
```

```python
scores = [40, 50, 80]

result = any(score >= 60 for score in scores)

print(result)
```

---

# 12. 객체 식별 관련 함수

## 12.1 id()

객체의 고유 식별값을 확인한다.

```python
a = 10
b = 10

print(id(a))
print(id(b))
```

```python
list1 = [1, 2, 3]
list2 = [1, 2, 3]

print(id(list1))
print(id(list2))
```

`id()`는 값이 같은지 비교하는 함수가 아니라, 객체가 메모리상에서 같은 객체인지 확인할 때 사용한다.

# 13. 실무에서 자주 쓰는 조합

## 13.1 입력받은 숫자 리스트 만들기

```python

```

입력 예시

```text
10 20 30 40
```

실행 결과

```text
[10, 20, 30, 40]
```

---

## 13.2 점수 합계와 평균 구하기

```python
scores = [80, 90, 70, 100]

total = sum(scores)
avg = total / len(scores)

print("합계:", total)
print("평균:", avg)
```

---

## 13.3 최댓값, 최솟값 구하기

# 14. 실습 문제

## 문제 1. 홀짝 판별

정수 하나를 입력받아 다음과 같이 출력하시오.

입력

```text
100
```

출력

```text
100 is even
```

예시 코드

```python
num = int(input())

if num % 2 == 0:
    print(num, "is even")
else:
    print(num, "is odd")
```

---

## 문제 2. 점수 합계와 평균

공백으로 구분된 점수 여러 개를 입력받아 합계와 평균을 출력하시오.

입력

```text
80 90 70 100
```

예시 코드

```python
scores = list(map(int, input().split()))

total = sum(scores)
avg = total / len(scores)

print("합계:", total)
print("평균:", avg)
```

---

## 문제 3. 가장 큰 값과 가장 작은 값

숫자 여러 개를 입력받아 가장 큰 값과 가장 작은 값을 출력하시오.

입력

```text
10 50 20 80 30
```

예시 코드

```python
numbers = list(map(int, input().split()))

print("최댓값:", max(numbers))
print("최솟값:", min(numbers))
```

## 문제 5. 생산량 합계 구하기

다음 생산량 리스트의 총 생산량, 평균 생산량, 최대 생산량, 최소 생산량을 출력하시오.

```python
qty_list = [100, 200, 150, 300, 250]
```

예시 코드

```python
qty_list = [100, 200, 150, 300, 250]

print("총 생산량:", sum(qty_list))
print("평균 생산량:", sum(qty_list) / len(qty_list))
print("최대 생산량:", max(qty_list))
print("최소 생산량:", min(qty_list))
```

# 15. 정리

Python Built-in 함수는 Python을 사용할 때 가장 먼저 익혀야 하는 기본 도구이다.

특히 초보자는 다음 함수들을 우선적으로 익히는 것이 좋다.

```python
print()
input()
int()
float()
str()
bool()
type()
len()
range()
sum()
max()
min()
sorted()
enumerate()
zip()
map()
filter()
all()
any()
```

이 함수들을 익히면 기본 입출력, 반복문, 리스트 처리, 간단한 데이터 분석까지 자연스럽게 연결할 수 있다.

# C언어 반복문 (for, while, do~while)

## 반복문이란?

반복문은 동일한 작업을 여러 번 수행할 때 사용하는 문법이다.

예를 들어 다음과 같은 상황에서 사용한다.

* 생산량 누적
* 센서 데이터 수집
* 제품 검사
* 메뉴 반복 출력

C언어의 대표적인 반복문은 다음과 같다.

| 반복문      | 특징               |
| -------- | ---------------- |
| for      | 반복 횟수가 정해진 경우    |
| while    | 조건이 참인 동안 반복     |
| do~while | 최소 1회 실행 후 조건 검사 |

---

# 1. for문

## 기본 문법

```c
for (초기식; 조건식; 증감식)
{
    실행문;
}
```

실행 순서

```text
초기식
↓
조건식 검사
↓
실행문 수행
↓
증감식 실행
↓
조건식 검사
↓
반복
```

---

## 예제 : 1부터 5까지 출력

```c
#include <stdio.h>

int main()
{
    int i;

    for(i = 1; i <= 5; i++)
    {
        printf("%d\n", i);
    }

    return 0;
}
```

실행 결과

```text
1
2
3
4
5
```

---

## 스마트팩토리 예제

제품 5개 검사

```c
#include <stdio.h>

int main()
{
    int result;
    int normal = 0;
    int defect = 0;

    for(int i = 1; i <= 5; i++)
    {
        printf("%d번째 제품 (정상:1, 불량:0): ", i);
        scanf("%d", &result);

        if(result == 1)
            normal++;
        else
            defect++;
    }

    printf("정상 제품 수 : %d\n", normal);
    printf("불량 제품 수 : %d\n", defect);

    return 0;
}
```

# 2. while문

## 기본 문법

```c
while(조건식)
{
    실행문;
}
```

조건식이 참인 동안 계속 반복한다.

---

## 예제 : 1부터 5까지 출력

```c
#include <stdio.h>

int main()
{
    int i = 1;

    while(i <= 5)
    {
        printf("%d\n", i);
        i++;
    }

    return 0;
}
```

---

## 스마트팩토리 예제

목표 생산량 달성

```c
#include <stdio.h>

int main()
{
    int target = 100;
    int production = 0;
    int input;

    while(production < target)
    {
        printf("생산량 입력 : ");
        scanf("%d", &input);

        production += input;

        printf("현재 생산량 : %d\n", production);
    }

    printf("목표 생산량 달성\n");

    return 0;
}
```

# 3. do~while문

## 기본 문법

```c
do
{
    실행문;
}
while(조건식);
```

특징

* 최소 1회는 반드시 실행된다.
* 메뉴 프로그램에 자주 사용된다.

---

## 예제 : 메뉴 프로그램

```c
#include <stdio.h>

int main()
{
    int menu;

    do
    {
        printf("\n");
        printf("1. 생산 시작\n");
        printf("2. 설비 상태 확인\n");
        printf("0. 종료\n");

        printf("선택 : ");
        scanf("%d", &menu);

        if(menu == 1)
        {
            printf("생산 시작\n");
        }
        else if(menu == 2)
        {
            printf("설비 상태 확인\n");
        }
        else if(menu == 0)
        {
            printf("프로그램 종료\n");
        }
        else
        {
            printf("잘못된 입력\n");
        }

    } while(menu != 0);

    return 0;
}
```

# 4. break문

반복문을 즉시 종료한다.

```c
#include <stdio.h>

int main()
{
    int temp;

    while(1)
    {
        printf("온도 입력 : ");
        scanf("%d", &temp);

        if(temp >= 100)
        {
            printf("위험 온도 발생\n");
            break;
        }

        printf("정상 상태\n");
    }

    return 0;
}
```

# 5. continue문

현재 반복을 건너뛰고 다음 반복으로 이동한다.

```c
#include <stdio.h>

int main()
{
    for(int i = 1; i <= 5; i++)
    {
        if(i == 3)
            continue;

        printf("%d번 제품 검사 완료\n", i);
    }

    return 0;
}
```

실행 결과

```text
1번 제품 검사 완료
2번 제품 검사 완료
4번 제품 검사 완료
5번 제품 검사 완료
```

---

# 6. 중첩 반복문

반복문 안에 반복문을 사용하는 구조

```c
#include <stdio.h>

int main()
{
    for(int line = 1; line <= 3; line++)
    {
        printf("라인 %d\n", line);

        for(int product = 1; product <= 5; product++)
        {
            printf("  제품 %d 검사\n", product);
        }
    }

    return 0;
}
```

# 핵심 정리

* for문 : 반복 횟수가 정해진 경우
* while문 : 조건이 참인 동안 반복
* do~while문 : 최소 1회 실행
* break : 반복 종료
* continue : 현재 반복 건너뛰기
* 중첩 반복문 : 반복문 안에 반복문 사용

# C언어 문자(Character)와 문자열(String)

## 문자와 문자열이란?

프로그램은 숫자뿐 아니라 문자와 문자열도 처리할 수 있어야 한다.

예를 들어

* 작업자 이름
* 제품 코드
* 설비 이름
* 생산 라인 이름
* 비밀번호

등을 저장할 때 사용한다.

---

# 1. 문자(Character)

문자(Character)는 하나의 글자를 의미한다.

C언어에서는 `char` 자료형을 사용한다.

---

## 문자 변수 선언

```c
char grade;
```

---

## 문자 저장

```c
char grade = 'A';
```

주의

```c
'A'
```

는 문자

```c
"A"
```

는 문자열이다.

---

## 문자 출력

```c
#include <stdio.h>

int main()
{
    char grade = 'A';

    printf("%c\n", grade);

    return 0;
}
```

실행 결과

```text
A
```

---

# 2. 문자 입력

```c
#include <stdio.h>

int main()
{
    char grade;

    printf("등급 입력 : ");
    scanf("%c", &grade);

    printf("입력값 : %c\n", grade);

    return 0;
}
```

---

# 3. ASCII 코드

컴퓨터는 문자를 숫자로 저장한다.

예를 들어

| 문자 | ASCII |
| -- | ----- |
| A  | 65    |
| B  | 66    |
| C  | 67    |
| a  | 97    |
| b  | 98    |
| c  | 99    |
| 0  | 48    |
| 1  | 49    |

---

## 문자와 숫자 출력

```c
#include <stdio.h>

int main()
{
    char ch = 'A';

    printf("%c\n", ch);
    printf("%d\n", ch);

    return 0;
}
```

실행 결과

```text
A
65
```

---

# 4. 문자열(String)

문자열은 여러 개의 문자가 모인 것이다.

예시

```text
SmartFactory
MES
PLC
Hello
```

---

## 문자열 선언

```c
char name[20];
```

20개의 문자를 저장할 수 있다.

---

## 문자열 저장

```c
char name[20] = "Hong";
```

---

## 문자열 출력

```c
#include <stdio.h>

int main()
{
    char name[20] = "Hong";

    printf("%s\n", name);

    return 0;
}
```

실행 결과

```text
Hong
```

---

# 5. 문자열과 문자 차이

## 문자

```c
char ch = 'A';
```

---

## 문자열

```c
char str[] = "A";
```

---

비교

| 종류  | 예제  |
| --- | --- |
| 문자  | 'A' |
| 문자열 | "A" |

# 6. 문자열 입력

## scanf 사용

```c
#include <stdio.h>

int main()
{
    char name[20];

    printf("이름 입력 : ");
    scanf("%s", name);

    printf("%s\n", name);

    return 0;
}
```

---

문제점

공백 입력 불가

입력

```text
Hong Gil Dong
```

결과

```text
Hong
```

만 저장된다.

---

# 7. fgets 사용

문자열 입력에 권장

```c
#include <stdio.h>

int main()
{
    char name[50];

    printf("이름 입력 : ");

    fgets(name, sizeof(name), stdin);

    printf("%s", name);

    return 0;
}
```

---

# 8. 문자열 길이

## strlen()

문자열 길이를 구한다.

```c
#include <stdio.h>
#include <string.h>

int main()
{
    char str[] = "SmartFactory";

    printf("%lu\n", strlen(str));

    return 0;
}
```

실행 결과

```text
12
```

---

# 9. 문자열 복사

## strcpy()

```c
#include <stdio.h>
#include <string.h>

int main()
{
    char source[20] = "PLC";
    char target[20];

    strcpy(target, source);

    printf("%s\n", target);

    return 0;
}
```

실행 결과

```text
PLC
```

---

# 10. 문자열 비교

## strcmp()

문자열 비교

```c
#include <stdio.h>
#include <string.h>

int main()
{
    char password[20] = "1234";

    if(strcmp(password, "1234") == 0)
    {
        printf("로그인 성공\n");
    }

    return 0;
}
```

---

주의

잘못된 방법

```c
if(password == "1234")
```

문자열은 반드시

```c
strcmp()
```

를 사용한다.

---

# 11. 문자열 연결

## strcat()

```c
#include <stdio.h>
#include <string.h>

int main()
{
    char str[50] = "Smart";

    strcat(str, "Factory");

    printf("%s\n", str);

    return 0;
}
```

실행 결과

```text
SmartFactory
```

---

# 12. 문자열 배열

여러 이름 저장

```c
char product[3][20] =
{
    "Motor",
    "Sensor",
    "PLC"
};
```

---

출력

```c
#include <stdio.h>

int main()
{
    char product[3][20] =
    {
        "Motor",
        "Sensor",
        "PLC"
    };

    for(int i=0; i<3; i++)
    {
        printf("%s\n", product[i]);
    }

    return 0;
}
```

# 14. 스마트팩토리 예제

설비 이름 검색

```c
#include <stdio.h>
#include <string.h>

int main()
{
    char machine[20];

    printf("설비 이름 입력 : ");
    scanf("%s", machine);

    if(strcmp(machine, "PLC") == 0)
    {
        printf("PLC 설비 선택\n");
    }
    else
    {
        printf("다른 설비 선택\n");
    }

    return 0;
}
```

# 15. 널 문자(Null Character)

문자열의 끝을 나타내는 특수 문자

```c
'\0'
```

예시

```text
H
e
l
l
o
\0
```

C언어는 `\0`을 만날 때 문자열이 끝났다고 판단한다.

# 핵심 정리

* `char`는 문자 1개 저장
* `%c`는 문자 입출력
* `%s`는 문자열 입출력
* 문자는 `'A'`
* 문자열은 `"A"`
* 문자열은 문자 배열로 구현
* 문자열 끝에는 `'\0'`이 존재
* `strlen()` : 길이 구하기
* `strcpy()` : 문자열 복사
* `strcmp()` : 문자열 비교
* `strcat()` : 문자열 연결
* 문자열 비교는 반드시 `strcmp()` 사용

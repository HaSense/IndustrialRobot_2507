# C언어 포인터(Pointer)

## 포인터란?

포인터(Pointer)는 변수의 메모리 주소를 저장하는 변수이다.

지금까지 우리는 변수에 값을 저장하였다.

```c
int num = 100;
```

메모리 상태

```text
주소      값
1000     100
```

변수 num은 값 100을 저장한다.

포인터는 값이 아니라 주소를 저장한다.

```text
주소      값
2000     1000
```

즉,

```text
1000 → num의 주소
2000 → num의 주소를 저장하는 포인터
```

---

# 왜 포인터가 필요한가?

포인터는 다음과 같은 경우 사용된다.

* 함수에서 원본 데이터 변경
* 배열 처리
* 문자열 처리
* 동적 메모리 할당
* 운영체제 및 임베디드 프로그래밍

---

# 메모리 구조 이해

프로그램이 실행되면 메모리에 데이터가 저장된다.

```c
int num = 100;
```

예시

```text
변수      주소      값
num      1000     100
```

---

# 주소 연산자 (&)

주소 연산자는 변수의 주소를 구한다.

## 예제

```c
#include <stdio.h>

int main()
{
    int num = 100;

    printf("%p\n", &num);

    return 0;
}
```

실행 결과

```text
0x61ff14
```

주소는 실행할 때마다 달라질 수 있다.

# 포인터 변수 선언

## 기본 문법

```c
자료형 *포인터변수;
```

예제

```c
int *ptr;
```

설명

```text
ptr은 정수형 주소를 저장하는 변수
```

---

# 포인터에 주소 저장

```c
int num = 100;
int *ptr;

ptr = &num;
```

메모리 구조

```text
num
주소 : 1000
값   : 100

ptr
주소 : 2000
값   : 1000
```

# 간접 참조 연산자 (*)

포인터가 가리키는 주소의 실제 값을 가져온다.

```c
*포인터변수
```

예제

```c
#include <stdio.h>

int main()
{
    int num = 100;

    int *ptr = &num;

    printf("%d\n", *ptr);

    return 0;
}
```

실행 결과

```text
100
```

```c
#include <stdio.h>

int main()
{
    int num = 100;

    int *ptr = &num;
```

실행 결과

```text
500
```

# 배열과 포인터

배열 이름은 첫 번째 요소의 주소이다.

```c
int data[5] = {10,20,30,40,50};
```

배열 이름

```text
data
```

는

```text
&data[0]
```

과 같다.

---

# 배열 주소 확인

```c
#include <stdio.h>

int main()
{
    int data[5] = {10,20,30,40,50};

    printf("%p\n", data);
    printf("%p\n", &data[0]);

    return 0;
}
```

동일한 주소가 출력된다.

---

# 포인터와 배열 접근

일반 방법

```c
data[2]
```

포인터 방법

```c
*(data + 2)
```

결과는 동일하다.

---

# 포인터 연산

```c
int data[5] = {10,20,30,40,50};

int *ptr = data;
```

| 표현식      | 결과      |
| -------- | ------- |
| ptr      | 첫 번째 주소 |
| ptr + 1  | 두 번째 주소 |
| ptr + 2  | 세 번째 주소 |
| *ptr     | 10      |
| *(ptr+1) | 20      |
| *(ptr+2) | 30      |

---

# 포인터 연산 예제

```c
#include <stdio.h>

int main()
{
    int data[5] = {10,20,30,40,50};

    int *ptr = data;

    for(int i=0; i<5; i++)
    {
        printf("%d\n", *(ptr+i));
    }

    return 0;
}
```

# 평균 계산 예제

```c
#include <stdio.h>

int main()
{
    int sensor[5] = {80,85,90,88,92};

    int *ptr = sensor;

    int sum = 0;

    for(int i=0; i<5; i++)
    {
        sum += *(ptr+i);
    }

    printf("평균 : %.2lf\n",
           (double)sum / 5);

    return 0;
}
```

# Call By Value

값 복사 방식

```c
void Change(int value)
{
    value = 100;
}
```

---

## 예제

```c
#include <stdio.h>

void Change(int value)
{
    value = 100;
}

int main()
{
    int num = 10;

    Change(num);

    printf("%d\n", num);

    return 0;
}
```

실행 결과

```text
10
```

원본이 변경되지 않는다.

---

# Call By Reference

주소 전달 방식

```c
void Change(int *value)
{
    *value = 100;
}
```

---

## 예제

```c
#include <stdio.h>

void Change(int *value)
{
    *value = 100;
}

int main()
{
    int num = 10;

    Change(&num);

    printf("%d\n", num);

    return 0;
}
```

실행 결과

```text
100
```

# Swap 함수

두 변수의 값 교환

```c
#include <stdio.h>

void Swap(int *a, int *b)
{
    int temp;

    temp = *a;
    *a = *b;
    *b = temp;
}

int main()
{
    int x = 10;
    int y = 20;

    Swap(&x, &y);

    printf("%d %d\n", x, y);

    return 0;
}
```

실행 결과

```text
20 10
```

# 다중 결과 반환

포인터를 이용하면 여러 결과를 반환할 수 있다.

```c
#include <stdio.h>

void Calculate(int a, int b,
               int *sum,
               int *mul)
{
    *sum = a + b;
    *mul = a * b;
}

int main()
{
    int sum;
    int mul;

    Calculate(10, 20,
              &sum,
              &mul);

    printf("합계 : %d\n", sum);
    printf("곱 : %d\n", mul);

    return 0;
}
```

# 스마트팩토리 예제

생산량과 불량률 계산

```c
#include <stdio.h>

void Calculate(int total,
               int defect,
               double *rate)
{
    *rate =
        (double)defect /
        total * 100.0;
}

int main()
{
    double rate;

    Calculate(1000,
              25,
              &rate);

    printf("불량률 : %.2lf%%\n",
           rate);

    return 0;
}
```

# 핵심 정리

* 포인터는 주소를 저장하는 변수
* `&` : 주소 구하기
* `*` : 주소가 가리키는 값 접근
* 배열 이름은 첫 번째 요소의 주소
* 포인터 연산으로 배열 접근 가능
* 문자열도 포인터로 처리 가능
* Call By Value는 값 복사
* Call By Reference는 주소 전달
* 포인터를 사용하면 함수에서 원본 데이터 수정 가능
* Swap 함수 구현 가능
* 여러 개의 결과를 반환할 수 있음

# C언어 구조체(Structure)

## 구조체란?

지금까지 우리는 하나의 변수에 하나의 값만 저장했다.

```c
int age = 20;
double score = 95.5;
char grade = 'A';
```

하지만 실제 프로그램에서는 하나의 대상을 표현하기 위해 여러 개의 데이터가 필요하다.

예를 들어 제품(Product)은 다음과 같은 정보를 가진다.

```text
제품번호
제품명
가격
재고수량
```

이처럼 서로 관련된 여러 데이터를 하나로 묶기 위해 사용하는 것이 구조체(Structure)이다.

---

# 구조체가 필요한 이유

예를 들어 제품 정보를 저장한다고 가정하자.

```c
int productId;
char productName[30];
int price;
int stock;
```

제품이 여러 개라면 변수 개수가 계속 증가한다.

구조체를 사용하면

```c
Product p1;
Product p2;
Product p3;
```

형태로 관리할 수 있다.

---

# 구조체 선언

## 기본 문법

```c
struct 구조체이름
{
    자료형 멤버1;
    자료형 멤버2;
    자료형 멤버3;
};
```

---

## 예제

```c
struct Product
{
    int id;
    char name[30];
    int price;
};
```

설명

```text
Product
 ├─ id
 ├─ name
 └─ price
```

---

# 구조체 변수 선언

## 기본 문법

```c
struct Product p1;
```

---

## 예제

```c
#include <stdio.h>

struct Product
{
    int id;
    char name[30];
    int price;
};

int main()
{
    struct Product p1;

    return 0;
}
```

# 멤버 접근 연산자 (.)

구조체 내부 데이터에 접근할 때 사용한다.

```c
구조체변수.멤버명
```

---

## 예제

```c
#include <stdio.h>

struct Product
{
    int id;
    char name[30];
    int price;
};

int main()
{
    struct Product p1;

    p1.id = 1001;
    p1.price = 5000;

    printf("%d\n", p1.id);
    printf("%d\n", p1.price);

    return 0;
}
```

# 구조체 배열

구조체도 배열로 만들 수 있다.

---

## 기본 문법

```c
struct Product products[10];
```

10개의 제품 정보를 저장

---

# 구조체 배열 초기화

```c
struct Product products[3] =
{
    {1001, "PLC", 500000},
    {1002, "Sensor", 20000},
    {1003, "Motor", 300000}
};
```

---

# 구조체 배열 출력

```c
#include <stdio.h>

struct Product
{
    int id;
    char name[30];
    int price;
};

int main()
{
    struct Product products[3] =
    {
        {1001, "PLC", 500000},
        {1002, "Sensor", 20000},
        {1003, "Motor", 300000}
    };

    for(int i=0; i<3; i++)
    {
        printf("%d %s %d\n",
               products[i].id,
               products[i].name,
               products[i].price);
    }

    return 0;
}
```

# 생산 이력 관리 실습

```c
#include <stdio.h>

struct Production
{
    int hour;
    int quantity;
};

int main()
{
    struct Production history[5] =
    {
        {1,100},
        {2,120},
        {3,140},
        {4,130},
        {5,150}
    };

    int total = 0;

    for(int i=0; i<5; i++)
    {
        printf("%d시간 : %d개\n",
               history[i].hour,
               history[i].quantity);

        total += history[i].quantity;
    }

    printf("총 생산량 : %d\n", total);

    return 0;
}
```

# 구조체와 포인터

구조체도 포인터로 접근할 수 있다.

---

## 구조체 포인터 선언

```c
struct Product p1;

struct Product *ptr;
```

---

## 주소 저장

```c
ptr = &p1;
```

---

# 구조체 포인터 접근

일반 접근

```c
p1.id
```

포인터 접근

```c
(*ptr).id
```

---

## 예제

```c
#include <stdio.h>

struct Product
{
    int id;
    int price;
};

int main()
{
    struct Product p1;

    struct Product *ptr = &p1;

    p1.id = 1001;
    p1.price = 5000;

    printf("%d\n", (*ptr).id);
    printf("%d\n", (*ptr).price);

    return 0;
}
```

---

# -> 연산자

구조체 포인터 접근을 쉽게 해준다.

---

## 문법

```c
ptr->id
```

위 코드는

```c
(*ptr).id
```

와 동일하다.

---

# -> 연산자 예제

```c
#include <stdio.h>

struct Product
{
    int id;
    int price;
};

int main()
{
    struct Product p1;

    struct Product *ptr = &p1;

    ptr->id = 1001;
    ptr->price = 5000;

    printf("%d\n", ptr->id);
    printf("%d\n", ptr->price);

    return 0;
}
```

# 설비 정보 관리 실습

```c
#include <stdio.h>
#include <string.h>

struct Machine
{
    int id;
    char name[30];
    int status;
};

int main()
{
    struct Machine machine;

    struct Machine *ptr = &machine;

    ptr->id = 1;

    strcpy(ptr->name, "Conveyor");

    ptr->status = 1;

    printf("설비번호 : %d\n",
           ptr->id);

    printf("설비명 : %s\n",
           ptr->name);

    printf("상태 : %d\n",
           ptr->status);

    return 0;
}
```

# 구조체를 함수로 전달

## 값 전달

```c
void PrintProduct(struct Product p)
{
    printf("%d\n", p.id);
}
```

---

## 주소 전달

```c
void PrintProduct(struct Product *p)
{
    printf("%d\n", p->id);
}
```

대형 구조체는 주소 전달이 효율적이다.

# 핵심 정리

* 구조체는 여러 자료형을 하나로 묶는 사용자 정의 자료형
* 멤버 접근은 . 연산자 사용
* 구조체 배열로 여러 객체를 관리할 수 있다.
* 구조체와 반복문을 함께 사용한다.
* 구조체 포인터는 구조체 주소를 저장한다.
* (*ptr).member 형태로 접근 가능
* ptr->member 형태가 더 많이 사용된다.
* 실제 프로그램에서는 구조체 + 배열 + 함수 + 포인터가 함께 사용된다.

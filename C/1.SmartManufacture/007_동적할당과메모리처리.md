# 동적 메모리 할당과 문자열 처리

---

# 학습목표

본 단원을 학습한 후 다음 내용을 수행할 수 있다.

* malloc()의 역할을 설명할 수 있다.
* 문자열 크기에 맞게 메모리를 동적으로 할당할 수 있다.
* strcpy()를 이용하여 문자열을 복사할 수 있다.
* 동적 메모리를 free()로 해제할 수 있다.
* 문자열을 결합하여 LOT 번호를 생성할 수 있다.

# 1. 왜 동적 메모리가 필요한가?

지금까지 사용한 문자열

```c

```

---

문제점

```text
항상 100바이트 사용
```

---

실제 입력

```text

```

---

필요한 공간

```text

```

---

하지만

```c

```

은 100바이트를 사용한다.

---

즉

```text

```

---

동적 할당을 사용하면

```text
필요한 만큼만

```

# 예제 1. 제품명 저장하기

사용자로부터 제품명을 입력받아 동적으로 메모리를 할당한 후 저장한다.

---

## 코드

```c
#include <stdio.h>
#include <stdlib.h>
#include <string.h>

int main()
{
    char temp[100];

    printf("제품명 입력 : ");
    fgets(temp, sizeof(temp), stdin);

    temp[strcspn(temp, "\n")] = '\0';

    char *product =
        (char *)malloc(strlen(temp) + 1);

    if(product == NULL)
    {
        printf("메모리 할당 실패\n");
        return 1;
    }

    strcpy(product, temp);

    printf("저장된 제품명 : %s\n", product);

    free(product);

    return 0;
}
```

---

## 실행 결과

```text
제품명 입력 : Motor

저장된 제품명 : Motor
```

# 예제 2. LOT 번호 생성하기

제품명과 생산일을 합쳐 LOT 번호를 생성한다.

---

## 코드

```c
#include <stdio.h>
#include <stdlib.h>
#include <string.h>

int main()
{
    char product[50];
    char date[20];

    printf("제품명 : ");
    scanf("%49s", product);

    printf("생산일 : ");
    scanf("%19s", date);

    int len =
        strlen(product) +
        strlen(date) +
        2;

    char *lot = (char *)malloc(len);

    if(lot == NULL)
    {
        printf("메모리 할당 실패\n");
        return 1;
    }

    strcpy(lot, product);
    strcat(lot, "_");
    strcat(lot, date);

    printf("LOT 번호 : %s\n", lot);

    free(lot);

    return 0;
}
```

---

## 실행 결과

```text
제품명 : Motor
생산일 : 20260623

LOT 번호 : Motor_20260623
```

# 핵심 정리

* malloc()은 실행 중 메모리를 할당한다.
* 문자열 길이에 맞게 메모리를 할당할 수 있다.
* strlen()으로 문자열 길이를 계산한다.
* strcpy()로 문자열을 복사한다.
* strcat()으로 문자열을 연결한다.
* malloc() 후에는 반드시 free()를 호출해야 한다.
* LOT 번호 생성은 제조업에서 매우 중요한 문자열 처리 예제이다.

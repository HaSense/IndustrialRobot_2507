# C 언어 연산자(Operator)

## 개요

연산자(Operator)는 변수나 상수에 대해 계산, 비교, 조건 판단 등의 작업을 수행하는 기호입니다.

예를 들어,

```c
int result = 10 + 20;
```

에서 `+`가 연산자입니다.

---

# 1. 관계 연산자 (Relational Operator)

관계 연산자는 두 값을 비교하여 결과를 참(True) 또는 거짓(False)으로 반환합니다.

C 언어에서는

- 참(True) = 1
- 거짓(False) = 0

으로 표현됩니다.

## 관계 연산자 종류

| 연산자 | 의미 | 예 |
|---------|---------|---------|
| == | 같다 | a == b |
| != | 같지 않다 | a != b |
| > | 크다 | a > b |
| < | 작다 | a < b |
| >= | 크거나 같다 | a >= b |
| <= | 작거나 같다 | a <= b |

---

## 예제 1

```c
#include <stdio.h>

int main()
{
    int a = 10;
    int b = 20;

    printf("%d\n", a < b);
    printf("%d\n", a > b);
    printf("%d\n", a == b);

    return 0;
}
```

실행 결과

```text
1
0
0
```

---

## 예제 2 : 점수 판별

```c
#include <stdio.h>

int main()
{
    int score = 85;

    if(score >= 60)
    {
        printf("합격\n");
    }

    return 0;
}
```

# 2. 논리 연산자 (Logical Operator)

여러 개의 조건을 결합할 때 사용합니다.

| 연산자 | 의미 |
|---------|---------|
| && | AND |
| \|\| | OR |
| ! | NOT |

---

## AND (&&)

모든 조건이 참이어야 참

```c
if(age >= 20 && age <= 60)
{
    printf("성인\n");
}
```

---

## OR (||)

하나만 참이어도 참

```c
if(score >= 90 || rank <= 3)
{
    printf("우수학생\n");
}
```

---

## NOT (!)

참과 거짓을 반전

```c
if(!isLogin)
{
    printf("로그인이 필요합니다.\n");
}
```

---

# 3. 산술 연산자 (Arithmetic Operator)

수학 계산에 사용합니다.

| 연산자 | 의미 | 예제 | 결과 |
|---------|---------|---------|---------|
| + | 덧셈 | 10 + 3 | 13 |
| - | 뺄셈 | 10 - 3 | 7 |
| * | 곱셈 | 10 * 3 | 30 |
| / | 나눗셈 | 10 / 3 | 3 |
| % | 나머지 | 10 % 3 | 1 |

---

## 예제

```c
int a = 10;
int b = 3;

printf("%d\n", a + b);
printf("%d\n", a - b);
printf("%d\n", a * b);
printf("%d\n", a / b);
printf("%d\n", a % b);
```

---

# 4. 증감 연산자 (Increment / Decrement)

변수 값을 1 증가 또는 감소시킵니다.

| 연산자 | 의미 |
|---------|---------|
| ++ | 1 증가 |
| -- | 1 감소 |

---

## 전위 증가

```c
int a = 10;
int b = ++a;
```

결과

```text
a = 11
b = 11
```

---

## 후위 증가

```c
int a = 10;
int b = a++;
```

결과

```text
a = 11
b = 10
```

---

# 5. 대입 연산자 (Assignment Operator)

변수에 값을 저장합니다.

| 연산자 | 의미 |
|---------|---------|
| = | 대입 |
| += | 더한 후 대입 |
| -= | 뺀 후 대입 |
| *= | 곱한 후 대입 |
| /= | 나눈 후 대입 |
| %= | 나머지 후 대입 |

---

## 예제

```c
int num = 10;

num += 5;

printf("%d\n", num);
```

결과

```text
15
```

---

# 6. 조건 연산자 (삼항 연산자)

if~else를 한 줄로 표현할 수 있습니다.

## 문법

```c
조건식 ? 참일때값 : 거짓일때값
```

## 예제

```c
int score = 80;

char* result = (score >= 60) ? "합격" : "불합격";

printf("%s\n", result);
```

결과

```text
합격
```

---

# 7. 비트 연산자 (Bitwise Operator)

비트 단위 연산에 사용합니다.

| 연산자 | 의미 |
|---------|---------|
| & | AND |
| \| | OR |
| ^ | XOR |
| ~ | NOT |
| << | 왼쪽 시프트 |
| >> | 오른쪽 시프트 |

---

## 예제

```c
int a = 5;
int b = 3;

printf("%d\n", a & b);
```

이진수

```text
5 = 0101
3 = 0011
-----------
    0001
```

결과

```text
1
```

---

# 8. sizeof 연산자

자료형 또는 변수의 크기를 구합니다.

```c
printf("%zu\n", sizeof(int));
printf("%zu\n", sizeof(double));
```

---

# 9. 주소 연산자 (&)

변수의 메모리 주소를 얻습니다.

```c
int num = 100;

printf("%p\n", &num);
```

---

# 10. 포인터 연산자 (*)

포인터가 가리키는 주소의 값을 참조합니다.

```c
int num = 100;
int* ptr = &num;

printf("%d\n", *ptr);
```

결과

```text
100
```

| 종류 | 주요 연산자 |

| 증감 | ++ -- |

# C언어 배열(Array)

## 배열이란?

배열(Array)은 동일한 자료형의 여러 개의 데이터를 하나의 변수 이름으로 관리하는 자료구조이다.

예를 들어 생산량 10개를 저장하려면 배열이 없다면 다음과 같이 작성해야 한다.

```c
int data1;
int data2;
int data3;
...
int data10;
```

배열을 사용하면 다음과 같이 관리할 수 있다.

```c
int data[10];
```

---

# 배열이 필요한 이유

다음과 같은 데이터를 저장할 때 사용한다.

* 센서 값
* 생산량
* 온도 데이터
* 검사 결과
* 학생 성적
* 제품 번호

---

# 1. 배열 선언

## 기본 문법

```c
자료형 배열이름[크기];
```

예제

```c
int score[5];
```

설명

```text
score[0]
score[1]
score[2]
score[3]
score[4]
```

총 5개의 정수를 저장할 수 있다.

---

# 2. 배열의 인덱스(Index)

배열은 0부터 시작한다.

```c
int score[5];
```

| 인덱스      | 의미    |
| -------- | ----- |
| score[0] | 첫 번째  |
| score[1] | 두 번째  |
| score[2] | 세 번째  |
| score[3] | 네 번째  |
| score[4] | 다섯 번째 |

---

## 값 저장

```c
score[0] = 90;
score[1] = 85;
score[2] = 100;
```

---

## 값 출력

```c
printf("%d\n", score[0]);
printf("%d\n", score[1]);
```

# 4. 배열 출력

```c
#include <stdio.h>

int main()
{
    int score[5] = {90,85,100,70,95};

    for(int i=0; i<5; i++)
    {
        printf("%d\n", score[i]);
    }

    return 0;
}
```

실행 결과

```text
90
85
100
70
95
```

# 5. 사용자 입력 받기

int main()

    int score[5];

    {
        printf("%d번째 점수 입력 : ", i+1);

    }

    for(int i=0; i<5; i++)
    {
        printf("%d\n", score[i]);

# 6. 배열과 합계 계산

## 예제

```c
#include <stdio.h>

int main()
{
    int score[5] = {80,90,100,70,85};
    int sum = 0;

    for(int i=0; i<5; i++)
    {
        sum += score[i];
    }

    printf("총점 : %d\n", sum);

    return 0;
}
```

실행 결과

```text
총점 : 425
```

# 7. 배열과 평균 계산

```c
#include <stdio.h>

int main()
{
    int score[5] = {80,90,100,70,85};
    int sum = 0;
    double avg;

    for(int i=0; i<5; i++)
    {
        sum += score[i];
    }

    avg = (double)sum / 5;

    printf("평균 : %.2lf\n", avg);

    return 0;
}
```

# 8. 최대값 찾기

```c
#include <stdio.h>

int main()
{
    int score[5] = {80,95,72,100,88};

    int max = score[0];

    for(int i=1; i<5; i++)
    {
        if(score[i] > max)
        {
            max = score[i];
        }
    }

    printf("최대값 : %d\n", max);

    return 0;
}
```

---

# 10. 스마트팩토리 예제

## 시간당 생산량 저장

```c
#include <stdio.h>

int main()
{
    int production[8];
    int sum = 0;

    for(int i=0; i<8; i++)
    {
        printf("%d시간 생산량 : ", i+1);
        scanf("%d", &production[i]);

        sum += production[i];
    }

    printf("총 생산량 : %d\n", sum);

    return 0;
}
```

# 12. 2차원 배열

행과 열로 구성된 배열

## 선언

```c
int data[3][4];
```

3행 4열

---

## 초기화

```c
int data[2][3] =
{
    {10,20,30},
    {40,50,60}
};
```

---

## 출력

```c
#include <stdio.h>

int main()
{
    int data[2][3] =
    {
        {10,20,30},
        {40,50,60}
    };

    for(int row=0; row<2; row++)
    {
        for(int col=0; col<3; col++)
        {
            printf("%d ", data[row][col]);
        }

        printf("\n");
    }

    return 0;
}
```

실행 결과

```text
10 20 30
40 50 60
```

---

# 13. 스마트팩토리 2차원 배열 예제

라인별 생산량

```c
#include <stdio.h>

int main()
{
    int production[3][4] =
    {
        {100,120,130,140},
        {90,100,110,120},
        {80,90,95,100}
    };

    for(int line=0; line<3; line++)
    {
        printf("라인 %d : ", line+1);

        for(int hour=0; hour<4; hour++)
        {
            printf("%d ", production[line][hour]);
        }

        printf("\n");
    }

    return 0;
}
```

# 핵심 정리

* 배열은 동일한 자료형을 여러 개 저장하는 구조
* 배열 인덱스는 0부터 시작
* 배열과 반복문은 항상 함께 사용
* 배열을 이용하면 대량의 데이터를 쉽게 관리 가능
* 최대값, 최소값, 평균 계산에 활용
* 2차원 배열은 표(Table) 형태 데이터를 저장
* 스마트팩토리에서는 생산량, 센서값, 검사결과 저장에 자주 사용

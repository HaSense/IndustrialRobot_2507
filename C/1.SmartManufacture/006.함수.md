# C언어 함수(Function)

## 함수란?

함수(Function)는 특정 기능을 수행하는 코드의 묶음이다.

우리가 지금까지 사용한 함수도 모두 함수이다.

```c
printf("Hello World\n");
scanf("%d", &num);
```

위의 `printf()`, `scanf()` 역시 함수이다.

---

# 함수를 사용하는 이유

함수를 사용하면 다음과 같은 장점이 있다.

* 코드 재사용
* 코드 중복 제거
* 유지보수 편리
* 프로그램 구조화

예를 들어 생산량 계산 코드가 여러 곳에서 필요하다면

```c
int GetProductionSum()
{
    ...
}
```

형태로 만들어 여러 번 사용할 수 있다.

---

# 함수의 구조

## 기본 문법

```c
반환형 함수이름(매개변수)
{
    실행문;
    
    return 반환값;
}
```

예제

```c
int Add(int a, int b)
{
    return a + b;
}
```

---

# 함수의 구성 요소

| 구성 요소  | 설명           |
| ------ | ------------ |
| 반환형    | 함수가 돌려주는 자료형 |
| 함수이름   | 함수의 이름       |
| 매개변수   | 함수에 전달되는 값   |
| return | 결과 반환        |

# 함수 호출 과정

```text
main()
 ↓
함수 호출
 ↓
함수 실행
 ↓
결과 반환
 ↓
main() 복귀
```

# 1. 반환값이 없는 함수

## 기본 문법

```c
void 함수명()
{
    실행문;
}
```

---

## 예제

```c
#include <stdio.h>

void Hello()
{
    printf("안녕하세요.\n");
}

int main()
{
    Hello();

    return 0;
}
```

실행 결과

```text
안녕하세요.
```

# 2. 반환값이 있는 함수

## 예제

```c
#include <stdio.h>

int Add(int a, int b)
{
    return a + b;
}

int main()
{
    int result;

    result = Add(10, 20);

    printf("%d\n", result);

    return 0;
}
```

실행 결과

```text
30
```

# 3. 매개변수(Parameter)

매개변수는 함수에 전달되는 값이다.

```c
int Add(int a, int b)
{
    return a + b;
}
```

여기서

```text
a
b
```

가 매개변수이다.

---

## 호출

```c
Add(10, 20);
```

전달값

```text
10 → a
20 → b
```

# 7. 스마트팩토리 예제

## 불량률 계산 함수

```c
#include <stdio.h>

double GetDefectRate(int total, int defect)
{
    return (double)defect / total * 100.0;
}

int main()
{
    double rate;

    rate = GetDefectRate(1000, 25);

    printf("불량률 : %.2lf%%\n", rate);

    return 0;
}
```

실행 결과

```text
불량률 : 2.50%
```

# 8. 배열과 함수

## 배열 합계 계산

```c
#include <stdio.h>

int GetSum(int data[], int size)
{
    int sum = 0;

    for(int i=0; i<size; i++)
    {
        sum += data[i];
    }

    return sum;
}

int main()
{
    int score[5] = {80,90,100,70,85};

    printf("합계 : %d\n",
           GetSum(score,5));

    return 0;
}
```

# 11. 전역 변수(Global Variable)

함수 밖에서 선언

```c
#include <stdio.h>

int count = 0;

void Increase()
{
    count++;
}

int main()
{
    Increase();

    printf("%d\n", count);

    return 0;
}
```

# 13. 재귀 함수(Recursive Function)

자기 자신을 호출하는 함수

## 예제

```c
#include <stdio.h>

void CountDown(int n)
{
    if(n == 0)
        return;

    printf("%d\n", n);

    CountDown(n - 1);
}

int main()
{
    CountDown(5);

    return 0;
}
```

실행 결과

```text
5
4
3
2
1
```

# 핵심 정리

* 함수는 특정 기능을 수행하는 코드의 묶음
* 코드를 재사용할 수 있다.
* 함수는 반환형, 함수명, 매개변수로 구성된다.
* `void` 함수는 값을 반환하지 않는다.
* `return`은 함수의 결과를 반환한다.
* 함수 선언과 정의를 분리할 수 있다.
* 배열도 함수의 매개변수로 전달할 수 있다.
* 지역 변수는 함수 내부에서만 사용된다.
* 전역 변수는 모든 함수에서 사용 가능하다.
* 큰 프로그램은 여러 함수로 나누어 작성한다.
